# W7S Docs

Public documentation for W7S, the open source platform that powers `w7s.cloud`.

## Development

```sh
npm install
npm run start
```

## Build

```sh
npm run check
```

The build output is deployed through W7S from the `build/` directory. The site is available at:

- `https://w7s-io.w7s.cloud/docs/`
- `https://docs.w7s.io/`
